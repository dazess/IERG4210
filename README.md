# Project Setup

## Tech Stack

- **Frontend:** React
- **UI Components:** ShadCN
- **Styling:** Tailwind CSS

## Prerequisites

- [Node.js](https://nodejs.org/)

## Installation

Install project dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

The application is available at `http://localhost:5173`.

## Project Structure (Phase 1)

### Styles
- `index.css`: Site-wide CSS styles
- `main.css`: Component-specific CSS styles

### Components
- `Footer.jsx`
- `Header.jsx`
- `HomePage-{category}.jsx`
- `ProductPage.jsx`
- `ShoppingCart.jsx`

### UI Library
- `ui/`: Directory containing installed ShadCN components (not writeen by me).
